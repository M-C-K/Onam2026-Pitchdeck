⚙ How to update pricing (click to expand)
  1. Click "Download pricing-data.xlsx to Edit" above, or pull the latest from the repo, to get the current file.
  2. Open it in Excel, LibreOffice, or even Google Sheets (upload it, edit, then re-export/download as .xlsx). It has two tabs:
  3. Offerings tab — one row per offering, columns:
    categoryiconidnamevaluemeaning
    Digital & Social📱social-media-promoSocial Media Promotions25000Your brand featured across...
    id should be unique per row (lowercase, no spaces) — used internally to track items.
  4. Settings tab — columns key / value, one row per setting:
    keyvalue
    gstRatePercent18
    eventNamePonn Onam 2026
    mainEventDatesSep 4, 5 & 6, 2026
    propertyExpoDatesJul 25 & 26, 2026
  5. Save the file as pricing-data.xlsx (same filename), replace it in the pitch-builder/ folder in the repo, then commit and push.
  6. ⚠ Only one person should edit at a time. This is a binary file in Git — if two people edit and push around the same time, the second push silently overwrites the first (no merge). Coordinate in chat before editing.
  7. Changes go live for everyone the next time they load the site, once your commit is deployed.
